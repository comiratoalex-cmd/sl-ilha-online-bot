# 🌴 Bot Discord — Ilha Online (Second Life)

Bot que renomeia canais do Discord mostrando:

- Pessoas online na ilha
- Pico máximo do dia

## 📦 Variáveis de Ambiente (Railway)

BOT_TOKEN=token do bot  
API_URL=url da API Google Apps Script  
GUILD_ID=id do servidor  
CHANNEL_ONLINE=id do canal de online  
CHANNEL_PEAK=id do canal de pico  

## ▶️ Start
\\\
npm install
npm start
\\\
